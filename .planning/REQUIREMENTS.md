## Out of Scope

| Feature | Reason |
|---------|--------|
| Site redesign / visual overhaul | Design is fine, problem is content-market fit |
| New comparison/alternative pages | Already have 13+11, focus on optimizing existing |
| Server-side rendering / framework migration | Static site works, no performance justification |
| Speculative content not backed by GSC data | Only build pages matching proven search demand |
| Free interactive tools | High JS/API effort, separate milestone |
